GIẢI THÍCH CẤU TRÚC THƯ MỤC

1. SRC/ - Thư mục chứa mã nguồn chính
   
   1.1. config/ - Cấu hình hệ thống
   - config.js: Cấu hình database (credentials, connection string)
   - constants.js: Các hằng số (status codes, role types, error messages)

   1.2. controllers/ - Xử lý business logic
   - auth.controller.js: Xử lý đăng nhập, đăng ký, xác thực
   - admin.controller.js: Quản lý hệ thống, verify tổ chức, thống kê
   - user.controller.js: Quản lý profile, cập nhật thông tin
   - charity.controller.js: Quản lý tổ chức từ thiện
   - donor.controller.js: Xử lý quyên góp, lịch sử đóng góp
   - campaign.controller.js: CRUD chiến dịch, thống kê
   - comment.controller.js: CRUD bình luận, đánh giá
   - upload.controller.js: Upload ảnh, tài liệu

   1.3. middlewares/ - Xử lý trung gian
   - auth.middleware.js: Verify JWT token
   - role.middleware.js: Kiểm tra quyền truy cập
   - upload.middleware.js: Xử lý upload file (multer)
   - validate.middleware.js: Validate input data
   - error.middleware.js: Xử lý lỗi tập trung

   1.4. models/ - Định nghĩa schema
   - index.js: Khởi tạo và kết nối các model
   - user.model.js: Schema người dùng chung
   - admin.model.js: Schema admin  
   - charity.model.js: Schema tổ chức
   - campaign.model.js: Schema chiến dịch
   - donation.model.js: Schema quyên góp
   - distribution.model.js: Schema phân phối
   - comment.model.js: Schema bình luận
   - otp.model.js: Schema mã OTP

   1.5. routes/ - Định tuyến API
   - index.js: Tập hợp và export routes
   - auth.routes.js: /api/auth/*
   - admin.routes.js: /api/admin/*
   - user.routes.js: /api/users/*
   - charity.routes.js: /api/charities/*
   - donor.routes.js: /api/donors/*
   - campaign.routes.js: /api/campaigns/*
   - upload.routes.js: /api/upload/*

   1.6. services/ - Xử lý bên thứ 3
   - auth.service.js: JWT, bcrypt
   - mail.service.js: Gửi email
   - sms.service.js: Gửi SMS
   - payment.service.js: Cổng thanh toán
   - upload.service.js: Cloud storage

   1.7. utils/ - Công cụ hỗ trợ
   - jwt.js: Generate/verify token
   - hash.js: Hash password
   - validate.js: Validation helpers
   - response.js: Format API response

   1.8. server.js - Entry point

2. TESTS/ - Unit tests
   - auth.test.js: Test authentication
   - admin.test.js: Test admin features
   - charity.test.js: Test charity features
   - campaign.test.js: Test campaign features

3. RESOURCE/ - Tài liệu
   3.1. Functional flow/ - Luồng xử lý
   - auth/: Luồng đăng nhập/đăng ký
   - admin/: Luồng quản trị
   - charity/: Luồng tổ chức
   - donor/: Luồng quyên góp
   - common/: Luồng chung

   3.2. Routes/ - API documentation
   - auth.routes.md: Auth API docs
   - admin.routes.md: Admin API docs
   - charity.routes.md: Charity API docs
   - donor.routes.md: Donor API docs
   - common.routes.md: Common API docs

4. CONFIG FILES
   - .env: Environment variables
   - .gitignore: Git ignore rules
   - package.json: Dependencies & scripts
   - README.md: Project documentation
